## 第1节:盒子模型
我们可以把一个html块元素看做是一个盒子，他具有如下css属性
``` css
selector{
    width:100px;            /*盒子的宽度*/
    height:100px;           /*盒子的高度*/
    margin:10px;            /*盒子的外边距*/
    padding:10px;           /*盒子的内编剧*/
    border:1px solid red;   /*盒子的边框*/
}
```
一个块元素有了这些属性就像是一个盒子，盒子里面可以摆放内容，内容也可以是另一个盒子，合理的运用盒子模型的这些css属性，就可以很轻松地实现我们想要的网页效果。
