## 第2节:jQuery入门
jQuery是JavaScript的一个库，可以让我们用更少的代码来实现更多的页面效果。如果希望使用jQuery，那么需要在html文档中引入jQuery库，引入方法如下:
``` html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
   
   <script src="script/jquery.js"></script> 
    <script>
        //这里可以使用jQuery提供的方法
    </script>
</body>
</html>
```
