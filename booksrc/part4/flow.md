## 第5节:语句流
程序有三种执行顺序：
* 顺序
* 分支
* 循环
之前写的示例代码都是按顺序执行的，下面我们来介绍分支和循环语句

#### 分支语句:if和if...else...
``` js
if(布尔值){
    //if语句内容
}
```
如上面示例代码所示，如果if后面的布尔值是true，则执行if语句的内容，如果是false则跳过不执行。

``` js
if(布尔值){
    //if语句内容
}else{
    //else语句内容
}
```
如上面示例代码所示，如果if后面的布尔值是true，则执行if语句的内容，如果是false则执行else语句后面的内容
