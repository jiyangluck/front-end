1-2-css基础

第2节：css基础

对于初学者来说，我们先将css写在html文件中，head标签下的script标签内，css语法

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style type="text/css">
        //css书写位置
    </style>
</head>
<body>

</body>
</html>

知道了样式书写位置，那用什么样的语法来设置样式呢

选择器{
    属性名:属性值
}

DEMO01:css选择器

首先应该学会使用最常用的集中选择器

  • id选择器
  • class选择器
  • 元素选择器
    #id{
  color:red;
}
.class{
  color:red;
}
h1{
  color:red;
}

DEMO02:css常用属性

刚才的例子使用了color属性，可以设置文字的颜色，接下来介绍更多常用的属性

#id{
    color:red;
    border: 1px solid red;
    width:500px;
    height:100px;
    text-align: center;
    line-height: 100px;
    font-size: 16px;
    background-color: #eee;
}
